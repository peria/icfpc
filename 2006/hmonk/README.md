```
advise <action> <advice_file> [<tests_file>]
```

### arith
`advise arith <file>` でテストされるのでそれが通ればOK。コメントにある通り参考実装の `Add` は左辺値、右辺値ともに値であるパターンしか書かれていないので、演算系が入力に来るパターンも網羅していく必要がある。
